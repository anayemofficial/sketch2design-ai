# Sketch2Design AI

Transform hand-drawn sketches into modern UI designs using AI.

## Features

- AI-powered sketch analysis
- Modern UI generation
- React + TypeScript + Vite
- Fast and responsive design

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```


## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
